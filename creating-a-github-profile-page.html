Creating a new Github Profile Page

lwx 19 Oct 2021 on Coding, Job

Github page

Github has a built in profile page which you can customize using markdown, and looks pretty good.

But like many others, this starts to feel lacking. For every github user, there is a free github pages which can be published at username.github.io.

For me, it’s at wxlai90.github.io - wxlai90 being my username.

A popular use case for this free landing page is to turn it into a profile page.

Previously, I hosted my SG Bus Arrivals app on this, which has now been moved to make space for a profile page.

To SPA or not to SPA

When deciding how best to build my new profile page, I decided upon a few rules to make things interesting.

  1. No using of templates
  2. No CSS libraries like bootstrap or material
  3. Keep it simple and clean

My main motivation behind setting those rules are to force myself to practise CSS.

I decided to do it with plain HTML & CSS. After all, we should use the tools that suit our use-case, and in this case HTML + CSS fits perfectly.

The bulk of a profile page definitely lies in the styling, which is something I am very bad at. I cannot do art to save my life and CSS is very much an art.

Nevertheless, I recognize the value of styling an app and have put in the effort to improve.

Creating this profile page from scratch is like a small test to myself, to evaluate how much I have improved and what areas to work on more.

Not too shabby

You can check out the final result here.

It was an impromptu decision to put my name next to my profile picture, and I tried to style it such that the initials of my name lines up but it looks kinda weird. I’ll leave improving in for a future iteration.

Overall, I am still happy with how it turned out, considering how bad I used to be at css.
